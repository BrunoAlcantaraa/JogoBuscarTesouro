# 🗝️ Jogo de Buscar Tesouro

Projeto desenvolvido em **Java 17** utilizando **Swing/AWT**, criado há aproximadamente **2 anos** durante meus estudos afim de compreender melhor conceitos de POO.

## 🎮 Sobre o Jogo

O objetivo do jogo é explorar o mapa em busca de **3 chaves** espalhadas pelo cenário. Cada chave é utilizada para abrir portas que bloqueiam o caminho até o tesouro.
Após coletar todas as chaves e desbloquear as portas, o jogador poderá acessar o baú e concluir a aventura.

## ✨ Funcionalidades

- Movimentação do personagem
- Sistema de colisão
- Coleta de itens
- Portas bloqueadas por chaves
- Mapa com exploração livre
- Efeitos sonoros e música
- Tela de conclusão do jogo

## 🚀 Como Executar

### Pré-requisitos

- Java JDK 17 ou superior
- Maven

### Executando

Clone o repositório:

```bash
git clone https://github.com/BrunoAlcantaraa/JogoBuscarTesouroJava.git
```

E execute diretamente a classe:

```bash

```
